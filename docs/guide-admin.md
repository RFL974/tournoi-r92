# Guide de l'organisateur

> ⚠️ Document en cours de rédaction. Il sera complété au fur et à mesure que les pages existent.

Ce guide explique, pas à pas, comment utiliser l'outil le jour du tournoi (et avant).

## Avant le tournoi
1. **Renseigner les réglages** (page admin → à venir) :
   - Horaires globaux : début, fin, pause déjeuner.
   - Pour chaque catégorie présente : terrains dédiés, taille de poule visée, format des matchs
     (mi-temps, durées, pauses, récupération).
2. **Saisir les équipes** avec leur catégorie.
3. Cliquer sur **« Générer poules et planning »** : l'outil répartit les équipes en poules
   (par catégorie) et calcule tous les horaires de match sans conflit.
4. Vérifier le planning généré.

## Pendant le tournoi
- **Saisie des scores** (page dédiée → à venir) : entrer le score de chaque match terminé.
- Les **classements** et le **live** se mettent à jour automatiquement.

## Pour les visiteurs
- **Mon planning** : chaque personne choisit son équipe et voit uniquement ses matchs.
- **Live** : classements par catégorie, derniers scores, possibilité de mettre des équipes en
  favori (étoile, stockée dans le navigateur), et lien de don HelloAsso.
