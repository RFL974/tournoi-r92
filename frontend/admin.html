Réinitialiser le tournoi

Remet le tournoi à zéro pour repartir d'une base vierge. Cela supprime définitivement :

  • toutes les catégories ;
  • toutes les équipes ;
  • toutes les poules et tous les matchs (planning + scores) ;
  • les infos du tournoi (nom, date, lieu, description, affiche).

Le tournoi repasse en masqué s'il était public.
✅ Sont conservés : les réglages « Horaires de la journée » et l'historique des saisons (page Perfs).
⚠️ Cette action est irréversible.
